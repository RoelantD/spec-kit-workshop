# Spec-Driven Development Workshop

A comprehensive hands-on workshop for learning spec-driven development with GitHub's Spec Kit toolkit. This workshop is part of the GitHub Copilot Certification Program (GH-300).

## üìö Workshop Contents

This workshop teaches the four-phase workflow of spec-driven development:

1. **Specify** - Define WHAT and WHY
2. **Plan** - Define HOW  
3. **Tasks** - Break down into implementable chunks
4. **Implement** - Build with AI assistance

### Chapters

- **Chapter 1**: Introduction to Spec-Driven Development
- **Chapter 2**: Setup & Prerequisites
- **Chapter 3**: The Specify Phase
- **Chapter 4**: The Plan Phase
- **Chapter 5**: The Tasks Phase
- **Chapter 6**: The Implementation Phase
- **Chapter 7**: Best Practices & Advanced Topics
- **Chapter 8**: Wrap-up & Next Steps
